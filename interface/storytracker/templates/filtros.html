Configuração de Parâmetros

Title x Text Importance: este parâmetro varia no intervalo [0,1]. Este parâmetro equilibra a pesquisa por páginas entre o termos pesquisados com o título e o texto dos documentos. O valor padão atribui total importância para o texto dos documentos, valor 1.

Months: este parâmetro define o ráio de pesquisa em mêses para um ponto no tempo. Supondo que a data inserida seja 2016-04-01, com raio de 3 mêses, a linha do tempo será construída a partir de 3 da data inserida e até 3 mêses após a data inserida, 2016-01-01 á 2016-06-31, respectivamente.
